## GreenApp

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Run Migration

```bash
# create new migrate
$ npm run create --name=NameMigrate

# genarate migrate
$ npm run generate --name=NameMigrate

# run migrate
$ npm run db:push

# revert migrate
$ npm run revert
```
